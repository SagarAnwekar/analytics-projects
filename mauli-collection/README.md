# 🛍️ Mauli Collection — Demand Forecasting Project

## Project Overview
**Customer Buying Behaviour and Demand Forecasting for Mauli Collection**
*Anwa, Bhokardan, Maharashtra | MBA Field Project 2025-26*

## Problem Statement
Mauli Collection, a local fashion retail business, needed to understand customer buying patterns and improve inventory management through demand forecasting.

## Research Methodology
1. **Primary Research** — Tick-box questionnaire survey design
2. **Data Collection** — Customer demographics, purchase frequency, product preferences
3. **Analysis** — Descriptive statistics, trend analysis, correlation
4. **Forecasting** — Moving average, weighted moving average, trend projection

## Tools Used
- Microsoft Excel (data cleaning, pivot tables, dashboards)
- Python (pandas, matplotlib, statistics)
- Survey-based primary data collection

## Key Deliverables
- [ ] Survey design and data collection
- [ ] Data cleaning and preprocessing
- [ ] Exploratory Data Analysis (EDA)
- [ ] Demand forecasting model
- [ ] Business recommendations report
